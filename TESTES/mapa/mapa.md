## 2. O que falta

### 1. Tratamento de Exceções (Erros)
- Aprender a lidar com erros para que seu programa não quebre inesperadamente.
- Conceitos: try, except, finally, raise.

### 2. Manipulação de Arquivos e Formatos de Dados
- Saber como ler e escrever em arquivos do sistema.
- Conceitos: open(), manipulação de arquivos .txt.
- Bibliotecas essenciais: json (muito usado no mercado) e csv.

### 3. Estruturas de Dados Embutidas (Avançado)
- Dominar as coleções do Python além das listas.
- Conceitos: Dicionários (dict), Tuplas (tuple), e Conjuntos (set). Pode ser que você tenha usado nas listas, mas vale um estudo aprofundado.

### 4. Ambientes Virtuais e Gerenciamento de Pacotes
- Aprender a isolar projetos e instalar bibliotecas de terceiros.
- Conceitos: venv (criação de ambientes virtuais), pip (instalação de pacotes) e o arquivo requirements.txt.

### 5. Consumo de APIs (Integração Web)
- Fazer seu código Python conversar com outros serviços na internet (ex: pegar cotação do dólar, consultar CEP).
- Conceitos: Biblioteca requests, métodos HTTP (GET, POST).

### 6. "Pythonices" (Recursos Específicos da Linguagem)
- Técnicas que deixam o código mais "Pythonico" e eficiente.
- Conceitos: List Comprehension (criar listas em uma linha), Decorators (@), e Generators.
